RISWIS Phase 3 Summary

Phase 3 extended RISWIS from a file-backed retrieval corpus into controlled governance-behavior testing.

What changed from earlier Phase 3 work

- manifest support for file-backed document paths was retained
- document embeddings continued resolving text from disk
- corpus expanded from initial controlled examples to a 12-document controlled set
- tier distribution was balanced across T1 / T2 / T3
- local embeddings were regenerated against the expanded corpus
- manifest-integrity verification remained enforced through hash matching

What Phase 3 tested

Phase 3 examined how explicit tier weighting changes retrieval outcomes across controlled semantic queries.

This moved RISWIS beyond a simple file-backed corpus test and into observable governance behavior analysis.

Observed behavior classes

- correct override cases where trusted documents overtook stronger lower-tier semantic matches
- policy-aligned override cases where governance promoted a higher-tier result in a defensible way
- neutral / reinforcement cases where semantic ranking and governance already aligned
- sensitivity cases where tier weighting appeared to over-correct semantic relevance

New visibility added

- raw semantic rank
- weighted rank
- rank delta per document

Why this matters

RISWIS can now show not only which document wins after governance weighting, but also how far documents move when policy is applied.

This makes governance effects visible instead of implicit.

Representative outcomes

- some queries showed higher-trust T1 documents overtaking stronger T2 or T3 semantic matches
- some queries showed no meaningful conflict between semantic relevance and trust weighting
- some queries exposed strong rank displacement, including cases where a document moved from raw rank 8 to weighted rank 2

Key result

Controlled governance behavior in RISWIS is now observable, classifiable, and auditable across a fixed retrieval corpus.

Phase 3 conclusion

Phase 3 established that RISWIS is not only capable of deterministic tier-weighted retrieval, but can also expose when governance intervention helps, stays neutral, or may over-correct semantic relevance.
