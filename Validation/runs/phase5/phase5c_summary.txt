RISWIS Phase 5C Summary
Multiplier Sensitivity Validation

Phase 5C tested how small changes in governance weighting alter final ranking once semantic similarity is already present.

Unlike Phase 5B, no new queries were introduced beyond the strongest threshold case already identified.

The purpose was to isolate multiplier strength as the only changing variable.

The same corpus, embedding cache, ranking equation, and query structure were preserved.

Only T1 multiplier strength changed.

The diagnostic query used was:

"feeling drained every day"

This query was selected because Phase 5B had already shown a clear semantic and governance conflict:

doc_111 ranked first in raw similarity but lost final weighted rank under baseline policy.

Four multiplier settings were examined:

T1 = 1.5
T1 = 1.3
T1 = 1.2
T1 = 1.1

T2 and T3 remained unchanged:

T2 = 1.0
T3 = 0.7

All Phase 5C runs used the same corpus, embedding cache, ranking equation, and query set, with only T1 multiplier strength adjusted.

What was observed:

1. At T1 = 1.5, T1 weighting strongly dominated final ranking.

   The raw T2 winner did not survive final rank 1.

2. At T1 = 1.3, T1 regained weighted rank 1 only narrowly.

   doc_110 weighted score: 0.640
   doc_111 weighted score: 0.622

   Margin: 0.018

3. At T1 = 1.2, the raw T2 winner survived final weighted rank 1.

4. At T1 = 1.1, semantic preservation increased further.

   A second T2 document entered weighted rank 3.

Concrete threshold table:

T1 Multiplier | Top Ranked Result | Behavior
1.5 | T1 override | strong governance dominance
1.3 | T1 narrowly wins | threshold edge
1.2 | T2 survives rank 1 | semantic survival begins
1.1 | T2 + second T2 | near-semantic regime

Threshold progression (visual reading):

1.5  ██████████  strong governance dominance
1.3  ████████░░  threshold edge
1.2  ██████░░░░  semantic survival begins
1.1  ████░░░░░░  near-semantic regime

Why this matters:

Phase 5C shows that RISWIS weighting is not simply active or inactive.

A multiplier shift of only 0.1 between 1.2 and 1.3 changed final rank ownership for the same retrieval event.

This means multiplier strength contains a measurable threshold zone.

What became visible:

At current baseline:

T1 = 1.5

Governance remains dominant.

But lower T1 values reveal a transition where semantic winners begin surviving final order.

This transition appeared between:

T1 = 1.2 and T1 = 1.3

Real-world interpretation:

A trust-weighted retrieval system may appear stable under one multiplier setting while changing materially under a very small policy adjustment.

RISWIS makes that threshold visible.

This means weighting choices are not neutral.

Small trust adjustments directly affect:

which source appears first

which source survives top-k

how much semantic evidence remains visible

Additional interpretation:

Phase 5C is the first RISWIS phase where governance influence became measurable as a transition band rather than simply observable as ranking pressure.

Lowering multiplier strength did not produce ranking instability.

Semantic ordering remained coherent while governance influence weakened gradually.

This suggests RISWIS weighting operates as controlled ranking pressure rather than random override.

Important caution:

The threshold observed here belongs to this corpus and query family.

Different corpora may produce different transition zones.

Operational interpretation:

1.5 = strong governance zone

1.2 – 1.3 = transition zone

1.1 = near-semantic zone

Baseline recommendation:

RISWIS baseline remains:

T1 = 1.5
T2 = 1.0
T3 = 0.7

Lower values in Phase 5C were sensitivity probes only and do not replace default governance policy.

Conclusion:

Phase 5C confirms that RISWIS contains a measurable multiplier threshold where semantic similarity and governance weighting exchange control of final rank order.

The semantic layer remains active.
The governance layer remains inspectable.
Threshold behavior is now directly observed.

Phase 5C completes the current multiplier sensitivity foundation for RISWIS Phase 5.
